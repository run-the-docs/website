# What it is

Run the Docs turns documentation into short, single-concept videos — each one built straight from the official docs. Every episode is rendered in two formats (a 4:5 feed cut and a 9:16 vertical) and published both self-hosted on this site and to the @RuntheDocs YouTube channel.

# How it runs

  • Source of truth — each episode is built from the concept's official documentation page.
  • Formats — every episode renders to both 4:5 (feed) and 9:16 (vertical).
  • Hosting — self-hosted on this site (Cloudflare Pages) and on YouTube @RuntheDocs.
  • Release — episodes are scheduled as a steady drip rather than dropped all at once.
